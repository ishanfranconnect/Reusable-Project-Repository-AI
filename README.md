# AI Project Generator

A full-stack application that generates project foundations based on user prompts.

## Tech Stack
- **Backend**: Java 17, Spring Boot 3.2.3, Spring Data JPA, MySQL
- **Frontend**: React, Vite, Axios, Modern CSS

## Setup

### Backend
1. Configure MySQL in `backend/src/main/resources/application.properties`.
2. Run `mvn spring-boot:run` in the `backend` directory.

### Frontend
1. Run `npm install` in the `frontend` directory.
2. Run `npm run dev` in the `frontend` directory.
3. Access at `http://localhost:5173`.
